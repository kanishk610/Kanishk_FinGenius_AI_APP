# Kanishk_FinGenius_AI_APP

🚀 FinGenius – AI Powered Personal Finance Assistant

An AI-driven personal finance web application designed to help students and young professionals manage money smarter.

📌 Problem Statement

Managing personal finances is difficult for students and young professionals due to:

Lack of financial literacy

No personalized financial guidance

Rising digital scams and fraud

Manual & inconsistent expense tracking

💡 Solution – FinGenius

FinGenius is an AI-powered financial assistant that:

Automatically tracks & categorizes expenses

Analyzes spending behavior

Provides personalized budgeting & investment advice

Detects potential scams in real time

Helps users achieve financial goals

It acts as a virtual financial advisor, not just a budgeting app.

✨ Features

📊 Smart Budget & Expense Tracking

🤖 AI-Based Financial Chatbot

📈 Investment Insights Dashboard

🚨 Real-Time Scam & Fraud Detection

🎯 Goal-Based Savings Planner

🔐 Secure Digital Document Vault

📉 Budget Health Score System
